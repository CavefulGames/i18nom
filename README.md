# English

## i18nom
I18n(Internationalization) library for Luau and Reactive UI library for Roblox

## Installation
Install via pesde (Supported environments: `luau`, `roblox`)
```sh
pesde add caveful_games/i18nom
```

### Supported Libraries
- `react-lua`
- `fusion-0.2`, `fusion-0.3`
- `vide`
