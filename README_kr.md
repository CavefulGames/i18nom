# 한국어

## i18nom
반응형 로블록스 UI 프레임워크와 Luau를 위한 현지화 라이브러리

## 설치
pesde로 설치 (지원되는 환경: `luau`, `roblox`)
```sh
pesde add caveful_games/i18nom
```

### 지원 라이브러리
- `react-lua`
- `fusion-0.2`, `fusion-0.3`
- `vide`

### 사용 예시
- 준비중

### 문서
- 문서는 아직 준비중입니다! `luau-lsp`를 사용하시면 Intellisense 자동완성에서 각 메소드들의 사용 방법을 확인할 수 있습니다.
